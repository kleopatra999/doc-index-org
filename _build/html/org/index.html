ownCloud Documentation Overview¶

Getting ownCloud¶

You can download and install ownCloud on your own Linux server, use the Web Installer to install it on shared Web hosting, try some prefab cloud images, or sign up for hosted ownCloud services. See the Get Started page for more information.

Reporting Documentation Bugs & How to Contribute¶

If you find an error or omission in any of the manuals, we welcome your bug reports and contributions. Please see howto_contribute for instructions.
